Get started

The GOV.UK Prototype Kit works on Mac, Windows and Linux.

It lets you make prototypes using code, so it’s helpful to have some knowledge of HTML before you start using it.

Install the Prototype Kit

If you’ve installed the Prototype Kit before, or are comfortable using Git and a terminal, you might want to use the quick start guide.

The Prototype Kit works on Mac, Windows and Linux.

It usually takes about 15 minutes to install the Prototype Kit. Steps vary slightly depending on which operating system you are using.

Choose between the:

  • Mac installation guide
  • Windows installation guide

Support

The Prototype Kit is maintained and supported by the GOV.UK Design System team at the Government Digital Service (GDS).

If you need help installing it, you can ask in the #prototype-kit channel on cross-government Slack or email govuk-design-system-support@digital.cabinet-office.gov.uk.
