Install the Prototype Kit

Introduction

This guide will take you through the steps to install the Prototype Kit. Installation usually takes about 15 minutes depending on how much you need to download.

If you get stuck, you can ask for help on the [#prototype-kit](https://ukgovernmentdigital.slack.com/messages/prototype-kit) channel on cross-government Slack or email govuk-design-system-support@digital.cabinet-office.gov.uk.

The Prototype Kit works on Mac, Windows and Linux. Depending on which department you work for, you might need admin access on your computer to install it yourself.

If you do not have admin access, you can ask your IT team to install it for you.
